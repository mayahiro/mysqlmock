# v0.1.1 Release Notes

`v0.1.1` は schema dump compatibility の bug fix release です

## Bug Fixes

- 複数 table で同じ MySQL index name を使う schema loading を修正しました
- MySQL-visible index name は table scoped のまま扱い、SQLite 内部では private index name に変換することで SQLite の schema-wide index namespace との衝突を避けます
- `SHOW KEYS` と `information_schema.statistics` は元の MySQL index name を返します
- `DROP INDEX`、`ALTER TABLE ... DROP INDEX`、`ALTER TABLE ... RENAME INDEX` は MySQL index name から内部 SQLite index name を解決します

## Upgrade Notes

config 変更は不要です
これまで複数 table で同じ index name が出て失敗していた schema file は、index name を変更せずに読み込めるようになります
